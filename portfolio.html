Aqui estão alguns dos projetos que desenvolvi e que podem ser encontrados no meu GitHub.

  • Sistema automatizado de alimentação para pets (Kedra pet)

    • Kedra Pet é um sistema automatizado de alimentação para pets, controlado por uma plataforma web. Desenvolvido como Trabalho de Conclusão de Curso (TCC) do curso Técnico em Informática da Escola Agrícola de Jundiaí – UFRN, o projeto integra hardware, software embarcado e uma interface web moderna para facilitar a vida de tutores de animais. Tecnologias utilizadas: HTML, CSS, JavaScript, Firebase Realtime Database (armazenamento em nuvem), Node.js (API + WebSocket).
